My-Flix API

This is the back-end portion of a full-stack movie database application. The backend uses Node.js and Express to create a REST API that is hosted on Heroku. The API also makes use of body-parser and mongoose and was tested with postman.

Meet App

This is a serverless PWA that incorporates Google's Calendar API to allow users to search for events occuring across in cities. OAuth2 is implemented to authorize users. Jest was used to incpororate a Test Driven Development workflow. It is hosted on gh-pages.
